# 禁止事项与必做清单

> 本章是全书的浓缩精华——红线和底线，建议反复阅读。

## 🔴 绝对禁止事项

### 喂养

| 禁止 | 原因 |
|------|------|
| ❌ 1 岁前喂蜂蜜 | 肉毒杆菌芽孢中毒风险 |
| ❌ 1 岁前喂鲜牛奶 | 蛋白质/矿物质过载损伤肾脏 |
| ❌ 6 个月前喂辅食 | 肾脏/消化系统未成熟 |
| ❌ 6 个月前喂水 | 影响奶量摄入 + 水中毒风险 |
| ❌ 喂米汤/清插曲 | 营养密度极低，干扰母乳 |
| ❌ 边吃边玩手机/看电视 | 干扰吸吮节律，形成不良习惯 |
| ❌ 强行灌药/灌奶 | 误吸窒息风险 |
| ❌ 奶瓶中加入米粉/麦片 | 过度喂养 + 呛咳风险 |
| ❌ 用奶瓶喂母乳时微波加热 | 营养破坏 + 烫伤 |

### 睡眠

| 禁止 | 做法正确 |
|------|---------|
| ❌ 同床睡 | ✅ 同房不同床，独立婴儿床 |
| ❌ 趴睡/侧睡 | ✅ 仰卧 |
| ❌ 床上放枕头/被子/毛绒玩具 | ✅ 睡袋代替被子 |
| ❌ 让宝宝睡 car seat/摇椅过夜 | ✅ 婴儿床 |
| ❌ 剧烈摇晃哄睡 | ✅ 轻摇、白噪音、嘘拍 |
| ❌ 长时间"落地醒"抱着不放下 | ✅ 趁迷糊时放下 |

### 护理

| 禁止 | 原因 |
|------|------|
| ❌ 把新生儿抛高高 | 摇晃婴儿综合征 → 脑损伤 |
| ❌ 用电吹风近距离热风吹屁股 | 烫伤 |
| ❌ 用棉签掏耳朵/鼻子 | 推进更深处 + 鼓膜/黏膜损伤 |
| ❌ 挤新生儿乳房（"挤奶核"） | 感染 |
| ❌ 挤"马牙"/上皮珠 | 感染 |
| ❌ 挑剪舌系带 | 出血/感染 |
| │ |
| ❌ 给新生儿戴项链/手链 | 勒颈/缠绕窒息 |
| ❌ 把宝宝单独留在浴缸/高处 | 溺水/坠落 |
| ❌ 让宝宝单独和宠物相处 | 不可预测的攻击 |
| ❌ 接触宝宝前不洗手 | 感染 |

### 环境

| 禁止 | 原因 |
|------|------|
| ❌ 在宝宝身边吸烟 | 二手烟/三手烟 → SIDS、哮喘 |
| ❌ 用蚊香/杀虫剂 | 呼吸刺激、中毒 |
| ❌ 室内温度过高 + 穿过多 | 过热 → SIDS |
| ❌ 长时间用加湿器不清洗 | 霉菌/细菌扩散 |
| ❌ 香水/强烈气味 | 呼吸道刺激 |

### 用品

| 禁止 | 安全替代 |
|------|---------|
| ❌ 婴儿防滑粉（爽身粉） | ✅ 液体爽身/护臀膏 |
| ❌ 防撞床围 | ✅ 空旷婴儿床 |
| ❌ 婴儿枕头 | ✅ 平躺不用枕 |
| ❌ 墩布式学步车 | ✅ 固定学步车或地垫 |
| ❌ 传统蜡烛包 | ✅ 上紧下松的睡袋 |
| ❌ 来路不明的中药偏方 | ✅ 循证医学 |

### 医疗

| 禁止 | 原因 |
|------|------|
| ❌ 给新生儿酒精擦浴退烧 | 经皮吸收中毒 |
| ❌ 自行使用抗生素 | 耐药性、菌群紊乱 |
| ❌ 用阿司匹林退烧 | 瑞氏综合征 |
| ❌ 2 个月以下自行用退烧药 | 必须就医 |
| ❌ 捂汗退烧 | 体温飙升、脱水 |
| ❌ 用偏方"挑针"治黄疸 | 感染/败血症 |
| ❌ 鹅口疮涂紫药水 | 现已不推荐，有更安全方案 |

## 🟢 必做清单

### 每日必做

- [ ] 按需哺乳 8~12 次（含夜间 1~2 次）
- [ ] 每次喂奶拍嗝
- [ ] 检查尿布 8~12 次，大便即换
- [ ] 清醒俯卧（Tummy Time）累计 ≥ 30 分钟
- [ ] 补充维生素 D 400 IU
- [ ] 开窗通风 2~3 次
- [ ] 脐带护理（未脱落前）
- [ ] 观察大便颜色/次数
- [ ] 适量互动：说话、眼神交流、抚触

### 每周必做

- [ ] 洗澡 2~3 次
- [ ] 称体重并记录
- [ ] 修剪指甲
- [ ] 检查全身皮肤（湿疹/红臀/间擦疹）
- [ ] 清洗/消毒奶瓶/吸奶器（如使用）

### 每月必做

- [ ] 儿保体检
- [ ] 测量身长/头围
- [ ] 疫苗接种（按计划）
- [ ] 大运动/精细运动发育评估
- [ ] 整理换季衣物

### 出生后关键节点

| 时间 | 必做事项 |
|-----|---------|
| 产后 1 小时 | 尽早 skin-to-skin、开奶 |
| 24 小时内 | 乙肝疫苗第 1 针 + 卡介苗 |
| 出院前 | 新生儿筛查（遗传代谢病、听力、CH、G6PD） |
| 7~14 天 | 脐带脱落、复查黄疸 |
| 2 周 | 体重恢复到出生体重 |
| 1 个月 | 体检 + 乙肝疫苗第 2 针 |

### 出院必带物品清单

**宝宝用品：**

- [ ] 出院衣（连体衣 + 包被）
- [ ] 纸尿裤 NB 号 2~3 片
- [ ] 湿巾/棉柔巾
- [ ] 安全提篮（车载必备）

**妈妈用品：**

- [ ] 出院衣（宽松舒适）
- [ ] 产褥垫/卫生巾
- [ ] 哺乳内衣 2~3 件

**证件资料：**

- [ ] 出生医学证明
- [ ] 疫苗接种本（绿本）
- [ ] 新生儿疾病筛查报告
- [ ] 出院小结/医嘱单
- [ ] 乙肝疫苗接种记录

## 🟡 灰色地带——不必教条

### 可以做的事（打破迷思）

✅ 哺乳期可以吃辣（观察宝宝反应）
✅ 哺乳期可以喝咖啡（每天 ≤ 200mg 咖啡因，约一杯）
✅ 哺乳期可以适量吃海鲜（避免高汞鱼类：鲨鱼、剑鱼、金枪鱼）
✅ 可以用手机/电脑（辐射不影响母乳，但注意保护眼睛休息）
✅ 可以运动（散步、瑜伽、慢跑，避免剧烈碰撞）
✅ 可以化妆/染发（选择正规产品，避免宝宝接触头发/面部）
✅ 宝宝可以出门（足月健康即可，避开人群密集）
✅ 可以开空调（保持室温 22~24°C，风口不对准宝宝）
✅ 清醒时可以俯卧（Tummy Time 强烈推荐）
✅ 3 个月以上可以短时间使用跳跳椅/健身架
✅ 可以用安抚奶嘴（降低 SIDS 风险，2~3 岁前戒除不影响牙齿）

### 不必焦虑的事

- 纯母乳 vs 配方奶：喂饱宝宝是第一目标，不必为此自责
- "睡眠训练"：每个家庭节奏不同，不必须
- "科学育儿" vs "经验育儿"：取两者之长
- 别人家孩子体重/身高：看自家生长曲线趋势，不比单点
- 网络上的"完美妈妈"：社交媒体是滤镜
- 偶尔的失控/崩溃：寻求帮助比硬撑更重要

## 📌 求助与支持

### 何时求助哺乳顾问

- 衔乳疼痛超过 1 周
- 宝宝体重下降/不增
- 乳房硬块、堵奶反复
- 不确定宝宝是否吃饱

### 何时求助儿科医生

- 发热（3 个月以下任何发热）
- 黄疸加重/不退
- 呕吐/腹泻/便秘异常
- 皮疹无法判断
- 发育担忧
- **任何让你直觉不安的情况——相信直觉**

### 家长心理健康

产后抑郁发生率高达 15~20%。如果你：

- 持续情绪低落 > 2 周
- 对宝宝提不起兴趣
- 有伤害自己或宝宝的念头
- 失眠（不是被宝宝吵醒的那种）
- 极度焦虑、无法享受任何事

**请立即寻求专业帮助。** 产后抑郁是疾病，不是矫情，可以治疗。
