# 换尿布与臀部护理

## 一、换尿布完整流程

### 频率

- 新生儿每天 8~12 片
- 每次喂奶前检查
- 有大便必须立即换
- 夜间只有大便才换（小便可等到天亮，避免完全唤醒）

### 步骤

**准备阶段**

- 洗手，准备用品：新尿布、护臀霜、温水棉柔巾/湿巾、隔尿垫
- 让宝宝平躺在安全平坦处

**清洁阶段**

1. 打开旧尿布，用尿布干净的上半部分先擦去大部分大便
2. **女宝**：从前向后擦（防止大便污染尿道口/阴道）
3. **男宝**：清理阴茎、阴囊皱褶，从上向下擦
4. 对折旧尿布垫在下面（防突然尿/屎突袭）
5. 用棉柔巾蘸温水擦净残留（湿巾含防腐剂，新生儿慎用）

**干燥阶段**

6. 用干纸巾或毛巾轻轻吸干水分（不要擦）
7. **等屁股完全晾干**（关键步骤！潮湿是红屁股根源）
8. 皱褶处、大腿根重点检查

**防护阶段**

9. 涂护臀霜形成隔离层（含氧化锌的为佳）
10. 涂抹范围：肛门周围 + 整个尿布接触区域
11. 不要涂太厚（影响皮肤呼吸）

**穿戴阶段**

12. 抬起宝宝双腿（抓住脚踝向上，不要抓脚趾提拉）
13. 垫入新尿布，前片从腿间穿过
14. **男宝**：阴茎朝下放置（防止尿向上射出）
15. 调整腿部防漏褶边（向外翻出！这是防漏关键）
- 每侧两指松紧：能塞进两根手指为宜
16. 粘扣对称粘贴，不要太紧

## 二、尿布疹（红屁股）防治

### 原因

- 潮湿（最主要）
- 摩擦
- 大便刺激（消化酶腐蚀皮肤）
- 真菌/细菌感染

### 分级

| 等级 | 表现 | 处理 |
|-----|------|------|
| 轻度 | 皮肤泛红 | 勤换 + 护臀霜 |
| 中度 | 红斑 + 轻微破皮 | 含氧化锌护臀霜 + 晾屁股 |
| 重度 | 糜烂、渗液、溃疡 | 就医，可能需抗真菌/激素药膏 |

### 治疗三板斧

**1. 勤换**

- 每 1~2 小时检查一次
- 大便不超过 5 分钟即换

**2. 干燥**

- 每次换尿布晾屁股 5~10 分钟
- 严重时每天赤屁股晒太阳/热风（低温档吹风机远距离吹）2~3 次

**3. 隔离**

- 涂含氧化锌护臀霜（如 Desitin、拜耳宝宝乐）
- 锌含量越高隔离效果越好（最高 40%）
- 严重时用凡士林做封包

### 进阶护理

- 湿巾换为温水棉柔巾
- 暂停使用婴儿皂
- 顽固不愈需就医排除真菌感染（念珠菌），用制霉菌素膏

## 三、纸尿裤选择

### 尺码选择

| 型号 | 体重 | 说明 |
|-----|------|------|
| NB | < 5kg | 新生儿专用 |
| S | 4~8kg | |
| M | 6~11kg | |
| L | 9~14kg | |
| XL | 12~17kg | |

> 不同品牌尺码有差异，以体重为主、腿围为辅。漏尿 = 小了，勒痕 = 大了。

### 选品标准

- 透气性（看面层材质）
- 吸水性（高分子吸水珠）
- 反渗情况（干爽度）
- 腰围弹性
- 是否有尿湿显示条
- 脐带专用款（脐带未脱落前用）

### 更换信号

- 有大便（立即）
- 尿湿显示条变色
- 明显沉甸甸
- 宝宝烦躁不安
- 每次喂奶前

## 四、大便观察指南

### 正常大便

| 阶段 | 特征 |
|-----|------|
| 胎便（出生 1~2 天） | 墨绿色、粘稠、沥青状 |
| 过渡便（3~4 天） | 黄绿色、较稀 |
| 母乳便（1 周后） | 金黄色、糊状/颗粒状、微酸味 |
| 配方奶便 | 淡黄色/棕黄色、成形、较臭 |

### 异常大便需就医

- **白色/灰白色便**：胆道闭锁（紧急！）
- **果酱样血便**：肠套叠（紧急！）
- **黏液脓血便**：细菌性肠炎
- **水样便频次明显增多**：病毒性腹泻
- **黑色柏油便**（非胎便期）：上消化道出血
- **鲜血便**：肛裂、过敏、肠息肉

### 母乳宝宝"攒肚"

- 部分 1~2 个月母乳宝宝多日不便
- 只要大便不干硬、排便不痛苦，属正常
- 不是便秘，是母乳吸收率高

## 五、如厕训练（预告）

- 通常 18~24 个月开始有信号
- 信号：对厕所有兴趣、尿布能保持干 2 小时、能表达便意
- 不建议过早（< 18 个月）训练，会造成心理压力
